Selamat Datang di Website Kampus Virtual



Kampus virtual merupakan sebuah proyek yang membuat suatu sistem pendidikan berbasis virtual yang bisa diakses dimanapun dan kapanpun. Pembuatan kampus virtual dilandasi oleh kemajuan teknologi yang memungkinkan pembelajaran dalam pendidikan dapat dilakukan dimana saja dan kapan saja, sehingga nantinya para peserta/mahasiswa dari kampus virtual dapat menjalankan perkuliahan dengan fleksibel.



Virtual campus is a project that creates a virtual-based education system that can be accessed anywhere and anytime. The creation of a virtual campus is based on technological advances that allow learning in education to be done anywhere and anytime, so that later participants/students can run their lectures flexibly.
